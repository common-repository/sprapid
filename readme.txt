== Description ==

With spRapid you can manage all your files at rapidshare.com. 
Optional you can access to your premium or collectors account and get all details. 

Upcomin feature: integration the files in your posts, translation files
	
== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
